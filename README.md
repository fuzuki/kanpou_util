■WEB版官報PDFを扱うためのライブラリ (2015.6) by fuzuki)


WEB版官報のタイトルやURLに簡単にアクセスするためのruby製簡易ツールです。
ダウンロードツールでは負荷軽減のため、アクセス毎に１秒間待機しています。
